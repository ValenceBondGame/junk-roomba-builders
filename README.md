# ガラクタ・ルンバ・ビルダーズ

Canvasで動作する、30秒制の2D見下ろし型部品回収ゲームです。外部画像・外部API・バックエンドは使用しません。

## 起動

Node.js 20.19以降（または22.12以降）を用意し、次を実行します。

```bash
npm install
npm run dev
```

表示されたローカルURLをブラウザで開いてください。

## 操作

- `WASD` / 矢印キー: 移動
- `E`: 車種ごとの最低必要数を満たしている場合に生成
- `Esc`: 一時停止 / 再開
- マウス: 設計図・完成車の選択

通常吸引は常時作動します。欲しい部品へ近づいて選んで回収してください。部品は小・中・大で吸引所要時間が異なり、回収された部品は5秒後にランダム地点へ復活します。回収箱は最大25個です。

## ルール

開始すると車種選択なしで30秒の回収が始まります。現在の部品に最も合う予想車種と完成度が常時表示され、`E` で複数台を生成できます。最低必要数は自転車5、ヘリコプター6、自動車7、飛行機8、電車10個です。終了時にプレイヤーとNPCそれぞれの最高車両を自動選択し、選ばれた車両自身の設計図に対して最大1,000点で採点します。同点時は正規部品数、ガラクタの少なさ、使用部品数の順で決着します。

## ビルド・公開

```bash
npm run build
```

`dist/` が静的成果物です。Cloudflare Pagesではビルドコマンドを `npm run build`、出力先を `dist` に設定してください。

## 検証

```bash
npm test
```

生成・割当・採点の純粋関数を簡易検証します。`?debug=1` をURLへ付けると、吸引範囲、NPC目的地、当たり判定、FPS、NPC回収数を表示します。
