# ガラクタ・ルンバ・ビルダーズ

Vite + Vanilla JavaScript + HTML Canvas で動く、部品回収・自動組立ゲームです。外部画像や外部APIは使用していません。

## 起動

```sh
npm install
npm run dev
```

## 操作

- `WASD` または矢印キー: 移動
- アイテムへ接近: 通常吸引（0.75秒）
- `P` / `Esc`: 一時停止・再開

部品が9個集まると、その内容に最も適した自動車・自転車・飛行機・ヘリコプター・電車を自動生成します。ラウンド終了後、生成した車両を1台選び、その車両自身のベース設計図に対する完成度でNPCと勝負します。

## 難易度

- EASY: 速度190、思考間隔0.45～0.75秒、探索420px、生成11個、判断ミス28%
- NORMAL: 速度245、思考間隔0.18～0.35秒、探索680px、生成9個、判断ミス10%
- STRONG: 速度285、思考間隔0.07～0.16秒、探索1200px、生成7個、判断ミス2%

選択は `grbDifficulty` として `localStorage` に保存されます。ラウンド開始時に固定され、途中では変化しません。

## アイテムと再出現

初期配置は40個（設定可能範囲36～44）です。プレイヤーとNPCの周囲へ各4個を確保し、残りを障害物や他アイテムと重ならないよう配置します。回収1個につき、ゲーム内時間で5秒後に回収品とは原則異なるランダムアイテムが安全な地点へ1個だけ再出現します。一時停止中はラウンド時間、NPC、吸引、再出現カウントのすべてが停止します。

## 検証

