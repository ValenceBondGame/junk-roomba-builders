# ガラクタ・ルンバ・ビルダーズ

Canvasで動作する、30秒制の2D見下ろし型部品回収ゲームです。外部画像・外部API・バックエンドは使用しません。

## 起動

Node.js 20.19以降（または22.12以降）を用意し、次を実行します。

```bash
npm install
npm run dev
```

表示されたローカルURLをブラウザで開いてください。

## 操作

- `WASD` / 矢印キー: 移動
- `Space` 長押し: 強力吸引（範囲と吸引力が上がり、移動速度が低下）
- `E`: 回収部品が6個以上なら生成
- `Esc`: 一時停止 / 再開
- マウス: 設計図・完成車の選択

通常吸引は常時作動します。部品は小・中・大で吸引所要時間が異なります。回収箱は最大25個です。

## ルール

自動車、自転車、飛行機、ヘリコプター、電車から目標設計図を選びます。同じステージのNPCと部品を奪い合い、30秒以内に複数台を生成できます。終了後に1台を選び、目標設計図との一致から最大1,000点で採点します。正規部品、共通部品、代用品、ガラクタ、欠品の順にガラクタ度が増えます。最高得点はブラウザのlocalStorageに保存されます。

## ビルド・公開

```bash
npm run build
```

`dist/` が静的成果物です。Cloudflare Pagesではビルドコマンドを `npm run build`、出力先を `dist` に設定してください。

## 検証

```bash
npm test
```

生成・割当・採点の純粋関数を簡易検証します。`?debug=1` をURLへ付けると、吸引範囲、NPC目的地、当たり判定、FPS、NPC回収数を表示します。
